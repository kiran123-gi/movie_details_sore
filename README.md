🎬 Movies CRUD Application

This project is a simple web application built using Django that performs CRUD operations (Create, Read, Update, Delete) on movie data.

🚀 Features:

* Add new movies using frontend form
* Store movie details in database
* Display movies on UI
* Update movie information
* Delete movies from database

🛠️ Tech Stack:

* Python
* Django
* HTML, CSS
* SQLite

📌 Purpose:
This project demonstrates how to integrate frontend with backend and perform database operations using Django ORM.

📂 Use Case:
Useful for beginners to understand CRUD operations and full-stack development basics.
